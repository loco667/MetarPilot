# Metar Pilot Support

If you need help with Metar Pilot, contact:

**Email:** MetarPilot@plek2.aero

## Frequently Asked

### Location is not updating
- On iPhone: Settings → Privacy & Security → Location Services → MetarPilot → **While Using the App**
- On Apple Watch: Watch app on iPhone → Privacy → Location Services

### Widget shows no data
- Open the app once and allow location access.
- Star an airport in the app for lock screen widgets.
- Wait a minute for widget refresh.

### How to report a bug
Please include:
- device model
- iOS/watchOS version
- app version
- airport code tested
- screenshot (if possible)
